# prismagram

instagram clone with express prisma and react native

## User Stories

- [o] Create account
- [o] Request Secret
- [o] Confirm Secret (Login)
- [o] Like/ Unlike a photo
- [o] Comment on a photo
- [o] Search user
- [o] Search post by location
- [o] Follow
- [o] Unfollow User
- [o] Edit my profile
- [ ] See user profile
- [ ] See my profile
- [ ] See the full photo
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
